# Roadmap
- Enhance the 3 core minigames according to the Driver Design Spec:
  - Precision Game: Dead Reckoning (oscillating reticle, 3 rings, rewards: upgrade card / double damage token / gold, penalty: Unfocused curse)
  - Timing Game: Resonance (6 pulses, varied rhythm. 120ms hit window. perfect/good/miss. rewards: zero-cost cards / card draw, penalty: Dissonance curse)
  - Speed Game: The Surge (4x4 grid. pattern recognition. threshold hits. rewards: draft pool expansion / gold / energy, penalty: Exhausted status)
- Refactor the Combat/Draft system to consume these nuanced rewards/penalties from Minigame results.
